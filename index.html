❓ FAQ

What exactly does this tool do?

The Agency Project Profitability Analyzer is a Python CLI tool that helps agencies track project budgets, log time entries with cost rates, calculate profit margins, and export interactive dashboards. It's like having a financial analyst in your terminal — no cloud accounts, no subscriptions, just pure data.

How is profit margin calculated?

Margin = (Total Billed − Total Expense) ÷ Total Billed × 100. Total Expense = (Hours × Cost Rate) + Overhead. Overhead is configurable (default 15%). The tool compares your actual margin against your target (default 40%) with 🟢 green/🟡 yellow/🔴 red health indicators.

Do I need to install anything besides Python?

No! The tool is zero-dependency — it uses only Python 3 standard library modules (json, csv, os, sys, datetime, pathlib, textwrap). Just download the .py file and run it.

Can I use this for multiple clients?

Yes. The tool supports unlimited projects and clients. Use --add-project to create new projects, then log time entries against each one. The portfolio-wide summary aggregates everything.

What output formats are supported?

Four formats: (1) Colorized console report, (2) JSON for programmatic use, (3) CSV for spreadsheet analysis, (4) Interactive HTML dashboard with visual bars, health tags, and portfolio summary.

Can I customize the settings?

Yes. Use --set hourly_bill_rate=175, --set internal_cost_rate=60, --set overhead_pct=20, or --set target_margin_pct=35. Settings are persisted locally.
